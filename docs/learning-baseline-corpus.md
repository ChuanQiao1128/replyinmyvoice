# Learning Baseline Corpus

Last updated: 2026-05-22

This corpus defines 100 synthetic or stripped baseline drafts for LearningOps evaluation. It is designed to test fact-preserving rewrite behavior across five product scenarios before new learning candidates are promoted into production guardrails, prompts, or tests.

No row below contains raw production text, private user content, account identifiers, or real user PII. Rows marked `learning_sample_derived` are fictionalized from aggregate `RewriteLearningSample` patterns already summarized in `docs/rewrite-memory-digest.md` and `docs/optimization-notes.md`, especially stock openings, overly safe tone, over-explaining, and drafts where the rewrite signal did not improve. Rows marked `hand_crafted_edge_case` are new fictional edge cases created for coverage.

The expected draft AI-like signal range is a pre-measurement target band for the rough draft only. It should be replaced by measured values when a full Sapling-backed evaluation run is executed.
